Body Text

Primary Text (18px, Regular 400): This is the standard paragraph text used throughout the website. It should be legible and comfortable to read for extended periods. The line height is set to 1.5 for better readability.

18px, Regular (400)
Karla

Secondary Text (16px, Regular 400): This smaller text is used for supporting information, captions, and footnotes. It's still legible but visually differentiated from the primary content.

Implementation Examples

CSS Variables

:root {
  --font-primary: 'Karla', sans-serif;
  --font-heading-alt: 'Inter', sans-serif;
  --font-heading-alt-2: 'Poppins', sans-serif;
  
  --weight-regular: 400;
  --weight-medium: 500;
  --weight-semibold: 600;
  --weight-bold: 700;
  
  --font-size-h1: 36px;
  --font-size-h2: 28px;
  --font-size-h3: 24px;
  --font-size-body: 18px;
  --font-size-secondary: 16px;
  --font-size-nav: 20px;
}

Example Usage

.page-title {
  font-family: var(--font-primary);
  font-size: var(--font-size-h1);
  font-weight: var(--weight-bold);
  letter-spacing: var(--letter-spacing-heading);
}

.doctor-description {
  font-size: var(--font-size-body);
  line-height: var(--line-height-base);
}

Responsive Behavior

On smaller screens (below 768px width), the typography scales down slightly to maintain readability: 

@media (max-width: 768px) {
  :root {
    --font-size-h1: 30px;
    --font-size-h2: 24px;
    --font-size-h3: 20px;
    --font-size-body: 16px;
    --font-size-secondary: 14px;
    --font-size-nav: 18px;
  }
}

This typography system ensures a professional and consistent look across the Doctor Appointment Booking Website.
